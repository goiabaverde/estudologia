Teoria sobre determinante de matriz

O determiante de uma matriz é uma forma de expressar toda uma matriz, facilitando o processo de encontrar a solução para um sistema linear de equações.

Para calcular o determiante de uma matrix 2x2 é possível usar o método de multiplicar os números da diagonal principal e subtrair do produto da diagonal segundária dessa mesma matriz.

>

Agora para calculaar o determinante de uma matriz 3x3 é usualmente utilizado a Regra de Sarrus que é um algoritmo que envolve multiplicações, somas e subtrações.

E para calcular o determinante de uma matrix 4x4 ou maior é possível utilizar o teorema de Laplace que é outro algoritmo que envolve escolher uma fila(coluna ou linha) e calcular para todos os elmentos dessa fila o seu cofator e multiplicar pelo próprio elemento após fazer isso basta somar todos os produtos gerados pelas operações anteriores.
